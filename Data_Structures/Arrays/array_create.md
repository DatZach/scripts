array_create
============

Returns an array populated by given arguments. 

    names = array_create("Charlie", "Juliett", "Mike", "Oscar", "Romeo", "Victor");
    
    //  names[0] == "Charlie"
    //  names[1] == "Juliett"
    //  names[2] == "Mike"
    //  names[3] == "Oscar"
    //  names[4] == "Romeo"
    //  names[5] == "Victor"

Note: GM:Studio has a 16 argument limit.
