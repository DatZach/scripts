bitwise_reverse32
=================

Returns the given 32-bit number with bits in reverse order.

    bits = 123456789;               //   123456789 ( 00000111 01011011 11001101 00010101 )
    rev = bitwise_reverse16(bits);  //  2830359264 ( 10101000 10110011 11011010 11100000 )
